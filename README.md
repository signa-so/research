# Signa Research

Evaluation artifacts for Signa's published technical reports
(https://docs.signa.so/research). One folder per report. Each contains the
frozen sample manifests (SHA-256 record-identifier digests, hashed before
any comparison ran), the frozen samples themselves (public register data),
and the per-record comparison results, so every number in a report can be
checked rather than taken on faith.

| Report | Folder |
|---|---|
| Deadlines You Can Docket Against (deadline verification) | `deadline-verification/` |

Source data is public trademark register data. No customer data appears in
any evaluation set.
